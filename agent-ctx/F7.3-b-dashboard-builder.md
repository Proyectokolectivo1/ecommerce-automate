## Page architecture

### Types
- `AuditUser`, `AuditLogWithUser`, `AuditListResponse`, `AuditStats`, `AuditFilters` — aligned with the API shapes from the spec.

### Constants
- `PAGE_SIZE = 15`
- `ENTITY_BADGE_CLASSES` — Order=emerald, User=amber, Shipment=teal, IntegrationSetting=violet, Transaction=zinc, PrintJob=rose; fallback `DEFAULT_ENTITY_BADGE` (zinc).
- `TEAL_BAR = '#14b8a6'` (teal-500) for actions chart, `VIOLET_BAR = '#8b5cf6'` (violet-500) for entities chart.

### Helpers
- `formatDateTimeSeconds(date)` — `dd/MM/yyyy HH:mm:ss` with `date-fns` + `es` locale (extends `lib/format.ts` `formatDate` which only goes to minutes).
- `prettyMetadata(raw)` — parses JSON string, falls back to raw on parse failure, pretty-prints with 2-space indent.

### Subcomponents
- `ActionBadge({ action, entity })` — monospace badge colored by entity (uses `ENTITY_BADGE_CLASSES[entity] ?? DEFAULT_ENTITY_BADGE`).
- `DetailCell({ metadata })` — renders `truncate(metadata, 40)` as the trigger; if truncated, wraps in `Tooltip` showing full pretty-printed JSON (`max-w-md whitespace-pre-wrap break-all font-mono`). Falls back to plain span if not truncated, or `—` if null.
- `UserCell({ user })` — `user.name` + `user.email` muted, or "Sistema" (with muted dot) if null.
- `ActionsChart({ data })` — Recharts horizontal `BarChart` (layout vertical), teal bars, Y axis `shortAction` (truncate 24), custom `ActionsTooltip` showing action + count.
- `EntitiesChart({ data })` — same pattern, violet bars, Y axis `shortEntity`, custom `EntitiesTooltip`.
- `ActionsTooltip` / `EntitiesTooltip` — custom popover content with `border bg-popover` styling.
- `EmptyChart({ message })` — `FileX2` icon + message when arrays are empty.
- `KpiSkeleton`, `ChartSkeleton`, `RowSkeleton` — loading skeletons.

### Main page
- `'use client'` top.
- State: `search`, `debouncedSearch`, `action` (default 'ALL'), `entity` (default 'ALL'), `startDate`, `endDate` (date inputs, empty string default), `page`.
- Debounce: 300ms `useEffect` for search input, resets page to 0.
- **Three useQuery hooks**:
  1. `audit-stats` (staleTime 30s) → `?stats=true`
  2. `audit-filters` (staleTime Infinity, fetch once) → `?filters=true`
  3. `audit-list` (placeholderData prev) → list with `search/action/entity/startDate/endDate/limit/offset`
- **Error toasts**: 3 `useEffect`s watching each query's `.error` → `toast.error` with description.
- **Layout** (`max-w-7xl space-y-6 p-4 md:p-6`):
  1. Header — `ScrollText` icon + "Auditoría" + subtitle "Registro de acciones y eventos del sistema".
  2. KPI row — grid `1→2→4` (sm→xl): Total eventos / Hoy / Últimas 24h / Últimos 7 días, icons `ScrollText / Calendar / Clock / History`. Shows `KpiSkeleton` × 4 while loading.
  3. Charts section — grid `1→2` (lg): two `Card`s with colored chip + title (`Acciones más frecuentes`, `Entidades más auditadas`), `ChartSkeleton` while loading, `EmptyChart` if empty, otherwise the chart.
  4. Filters + Table section — `Card`-styled container:
     - **Filters bar** (`border-b p-4`): Search input (with X clear button when not empty), Action Select (ALL + distinct actions from filters endpoint, items in monospace), Entity Select (ALL + distinct entities), date range (Desde/Hasta `Input type="date"`), "Limpiar" button when `hasActiveFilters`.
     - **Table** (`overflow-x-auto`): columns Fecha / Acción (badge) / Entidad (entity + entityId truncated 18, monospace) / Usuario / IP (hidden md, monospace) / Detalle (DetailCell). `RowSkeleton` × 8 while loading; empty state with `Inbox` icon, contextual message, and "Limpiar filtros" button when filters active.
     - **Updating badge** (top-right) when `listIsFetching && !listIsLoading` — "Actualizando…" with spinner.
     - **Pagination** (`Separator` + `p-3`): "Mostrando X–Y de N evento(s)" + Anterior / Pág. N / Siguiente buttons.
