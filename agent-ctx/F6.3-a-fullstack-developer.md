## Decisiones técnicas
- **Mutations**: primer intento usaba `buildMutation()` en loop → error `react-hooks/rules-of-hooks`. Refactor: `mutationOptions(config)` factory pura + 4 `useMutation` a top-level + diccionario `mutations`.
- **Reset página**: reemplacé `useEffect(() => setPage(0), [typeFilter])` (error `react-hooks/set-state-in-effect`) por `handleTypeChange` handler pasado al `onValueChange` del Select.
- **Markdown styling**: `mdComponents` object con clases tailwind para h2/h3/p/ul/ol/li/strong/a/blockquote/code/pre/table/thead/th/td/hr.
- **Truncate**: `truncateMarkdown` corta en último espacio antes del límite (200 chars) para no romper palabras.
- **TanStack Query**: stats (staleTime 30s), insights (placeholderData prev, queryKey con typeFilter+page). On mutation success: setPage(0) + invalidate ['ai','insights'] + ['ai','stats'].
